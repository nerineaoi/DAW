- Defining and instantiating classes and objects  
- Attributes, constructors, and methods  
- Encapsulation and access modifiers  
- Inheritance (`extends`) and method overriding (`@Override`)  
- Polymorphism and class hierarchies  
- Relationships between classes (composition, aggregation)
